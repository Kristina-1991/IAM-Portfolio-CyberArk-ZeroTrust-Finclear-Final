### Flux détaillés (les 8 flux que le recruteur veut entendre en entretien)

1. Employee lifecycle → Workday → SailPoint → Entra ID / AD (already live)
2. Account discovery → CyberArk DNA → toutes les cibles ci-dessus (new)
3. Semi-automated onboarding → Discovery → Pending Accounts → Owner assignment → Safe + rotation (BR-01)
4. JIT elevation → User demande via PVWA → dual approval → session 1-8 h → auto-revoke (BR-02)
5. Session recording → 100 % sessions via PSM → indexed + searchable → export Splunk (BR-03)
6. Password rotation → CPM policies (90 jours ou post-use) sur toutes les cibles
7. De-provisioning → SailPoint termination event → CyberArk API → disable + quarantine safe
8. Evidence & audit → CyberArk → Splunk (real-time) + Sentinel (cloud events)
