# Pilot OT/SCADA (Weeks 13-14 + lessons learned)

**Scope** → 250 critical devices (Siemens, Rockwell) – air-gapped environment

**Objectives**
- Validate custom connectors for legacy protocols  
- Test PSM in air-gapped mode  
- Document lessons learned for full rollout 2026 Q2

**Key Activities**
| Activity                                 | Owner        | Duration |
|------------------------------------------|--------------|----------|
| Discovery with restricted DNA agent      | OT team      | 1 week   |
| Deploy dedicated air-gapped PSM          | CyberArk SME | 2 weeks  |
| Onboard 50 highest-risk accounts         | PAM Admin    | 1 week   |
| Test JIT + recording                     | OT + IAM     | 1 week   |
| Lessons learned workshop + report        | IAM BA       | 1 week   |

**Success Criteria**
- ≥ 95 % accounts onboarded without production impact  
- Full session recording in isolated environment  
- Go/No-Go decision for full OT rollout in 2026

**Outcome** → Validated technical feasibility + governance model for OT.
